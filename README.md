# sudoku-react-app

A simple web application for playing Sudoku

## Dependencies

1. NPM (https://www.npmjs.com/get-npm)

## Getting started

1. Clone this repo
1. Change to the root directory of the app
2. Run `npm install`
3. Run `npm start`

## Launching locally

To launch the app locally simply run `npm start` from the root directory of the app.
